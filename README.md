# **Challenge Alura Store**

Trabajo final del Challenge Alura Store del curso de Practicando Python para Data Science de ALURA Latam | ONE Oracle Next Project. Se basa en el concepto Challenge Based Learning, es decir, un aprendizaje centrado en desafíos que Apple ayudó a crear y que se basa en 3 pilares: Compromiso - Investigación - Actuación.

Con esto, se buscó poner en práctica conceptos iniciales de análisis de datos con Python, utilizando bibliotecas nativas de Python para la manipulación/extracción de archivos y la visualización de datos.

Proyecto **finalizado**.

## **Objetivo del análisis realizado**

Se buscó brindarle al Sr. Juan, dueño de la cadena de tiendas Alura Store, una recomendación basada en datos de cual tienda se debería vender para iniciar un nuevo emprendimiento. 

## **Proyecto**

### Estructura

El proyecto se compone de dos partes: el análisis de datos, utilizando Python y sus respectivas bibliotecas, y el informe final, donde se detalla cuál tienda es la que se recomienda vender. Junto a este último, se pueden observar gráficos que apoyan la decisión de manera ilustrativa. 

### Gráficos e insights

Puntualmente, en este proyecto encontrarás el análisis de:

* Los ingresos totales de las tiendas.
* Las categorías de productos más y menos vendidas.
* Las calificaciones promedio de los clientes por tienda.
* Los productos más y menos vendidos.
* El coste de envío promedio para cada tienda.

### Instrucciones

Para ejecutar un archivo .ipynb, las formas más comunes son usar Jupyter Notebook/Lab localmente, VS Code, o Google Colab en la nube.

* Google Colab: sube el archivo a Google Colab (o Google Drive), abre el archivo y haz clic en el botón de reproducción en las celdas.

* Jupyter Notebook/Lab: inicia el servidor de Jupyter desde su terminal (jupyter notebook), navegue hasta el archivo en el navegador web y haga clic para abrirlo. Una vez abierto, use "Cell" > "Run All" o presione Shift + Enter para ejecutar celdas individualmente. 

* Visual Studio Code (VS Code): abre el archivo en VS Code (requiere la extensión de Python/Jupyter) y haz clic en "Run All" o ejecuta celda por celda.
